### 1. 使用 xargs rm 

```
ls | grep abcd | rm  //错误用法
```



rm doesn't accept input from `stdin`. You'll need to do something like

```
ls | grep abcd | xargs rm
```

但是遇到不规则符号的文件有可能删除不了.



### 2. 使用 find exec

可以删除不规则符号文件:
```
find . -name "*td*" -exec rm -f {} \;
```
