filebeat收集nginx的json格式日志

发表于 2021-01-23 更新于 2021-02-21 分类于 2-linux系统 ， elk 阅读次数： Valine：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

为了便于利用 ELK日志平台收集展示 Nginx 的日志，可以将 Nginx 的日志改成 json 的格式.

1. 配置

1.1 修改 nginx 配置

vim /etc/nginx/nginx.conf

通过浏览器访问, 可以看到 json 输出的日志

1.2 修改 filebeat 配置

vim /etc/filebeat/filebeat.yml

1.3 filebeat 服务

sudo vi /lib/systemd/system/filebeat.service

systemctl start filebeat
